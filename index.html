Cipher Quick Reference

Here's what each encoding method does:

Base64 Standard encoding for binary-to-text (most common in attacks)
Hexadecimal Converts each byte to two hex digits (0-9, A-F)
URL Encoding Encodes special characters for URLs (%20 = space)
ROT13 Shifts letters by 13 positions (A‚ÜíN, B‚ÜíO)
Binary Converts text to 1s and 0s (8 bits per character)
ASCII Shows the decimal value of each character (A=65)
Morse Code Classic dots and dashes (. = dit, - = dah)
Reverse Simply reverses the string backwards
Caesar Shifts all letters by 3 (A‚ÜíD, B‚ÜíE)
Atbash Reverses the alphabet (A‚ÜíZ, B‚ÜíY)
